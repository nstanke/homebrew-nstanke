Homebrew: Zim Desktop Wiki
===========================

## Prerequesits

* Homebrew
* Homebrew Cask

## Add tap

`brew tap nstanke/zim`

## Install Zim

`brew install zim-wiki`

## Install wrapper App

`brew cask install zim-app`
